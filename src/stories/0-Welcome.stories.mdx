# Installation
<Story name="Installation"></Story>

Install with yarn:
```
yarn add react-circular-progressbar
```
or npm:
```
npm install --save react-circular-progressbar
```

# Props
<Story name="Props"></Story>

props description here

| *Names* | *Description* | *Default Value*  | *Type* |
| ------ | ------ | ------ | ------ |
| size | Circle height & Width | `180` | Int |
| strokeWidth | Circle Stroke width |  `15`  | Int |
| strokeWidthBg | Circle Stroke Background width |  `15`  | Int |
| fillColor | Stroke Fill Color |  `#288feb`  | String |
| emptyColor | Stroke (empty) Background Fill Color |  `#288feb`  | String |
